Projects

Buns on the Run

Buns on the Run was my first website that I created for a school project. I used HTML, CSS and Dreamweaver to create the site. I designed the entire site including their logo which they still use on their YouTube channel. One of the requirements was to create a template after the homepage was created to keep from having to recreate the same elements that would be on all the additional pages, such as the logo, navigation links and the footer.

One of the more difficult parts of the project was trying to determine how to layout the steps of the recipes along with the images. The best way I felt to display this information to the users was to implement a pure CSS slideshow of the images numbered corresponding to the step they would be in following the recipe.

One of my favorite parts of the project besides the CSS slideshow would have to be on the recipes page. I chose that it would be best to show the different types of recipes they do by categories. As you hover over the images, CSS is used to display each category name for those recipes as displayed in the images of the website.

Dimes Times

Dimes Times was a site created for a project while I was in school. A friend was interested in having a site designed to showcase his artwork and books that he's written.

I designed this site with HTML and CSS later on revisited the project to practice making a site that was currently made for desktops to be mobile responsive using media queries.

One of the requirements for the project was to use a table in the design of one of the pages. I felt it was best to use the table to show the books that were for sale and providing links if the users were interested in purchasing them. The table design became difficult when making the site mobile responsive but also taught me a lot of how to manipulate elements while trying to get the perfect design for a mobile site.

One of my favorite parts of Dimes Times was my decision to make the whole site grayscale. While doing this I felt it would be best to make all of the images grayscale using CSS but also letting the user have the option to see the true representation of the artist's work by hovering of the images to view them in color.

In the end I actually believe the mobile site is better than the desktop site, but I'll leave that for you to judge.

Yama Sushi Bar

The Yama Sushi Bar website was created as a school project for a Dreamweaver class as well as the logo that is used on the site. We were told to start with a basic template and add in our own CSS to create our own design. One of the main topics discussed in the class and which we had to implement in this project was keeping in mind that the site would have to work on mobile devices.

The most difficult part in this project was most likely the menu page. I really liked how it was designed for desktops with all of the images of the food surrounding the menu but was really questioning how I would keep this design for mobile devices. I came to the conclusion that I wouldn't be able to show as many images and still have the content be easily accessible, so I felt that removing the images would be the best choice and to have the menu fill the width of the page.

I was really happy with the outcome of this project on desktops as well as on mobile devices
